什么是Shadowrocket？

Shadowrocket是一款iOS平台上的网络代理工具，专门为用户提供科学上网服务。它支持多种代理协议，如Shadowsocks、Vmess、Trojan等，能够有效帮助用户突破网络封锁，访问被限制的网站。无论是需要全局代理，还是希望自定义规则来灵活控制哪些流量走代理，Shadowrocket都能满足各种需求。

Shadowrocket的主要功能

1. 全局代理模式

Shadowrocket的全局代理模式允许用户将所有网络流量通过代理服务器转发。这意味着，您在使用任何应用时，所有网络请求都会自动通过代理进行加密和转发，从而实现科学上网。这一功能对于需要访问被封锁网站的用户来说至关重要。

2. 多种协议支持

Shadowrocket支持多种代理协议，包括Shadowsocks、Vmess、Trojan等。这些协议各具特色，用户可以根据需要选择最适合的协议。例如，Shadowsocks以其轻量、快速而受到广泛青睐，而Vmess协议则在隐私保护和抗干扰方面表现出色。通过支持多种协议，Shadowrocket能够提供更加灵活和高效的上网体验。

3. 自定义规则设置

Shadowrocket的自定义规则设置是其一大亮点。用户可以根据自己的需求设置访问规则，灵活地控制哪些流量走代理，哪些流量直连。例如，您可以设置某些特定的网站（如社交媒体平台）走代理，而其他网站（如新闻网站）则直接连接。这一功能为用户提供了更高的灵活性和控制权。

4. 流量统计与监控

为了帮助用户更好地管理网络流量，Shadowrocket还提供了实时流量监控和统计功能。用户可以查看各个应用的流量使用情况，实时掌握网络使用状况。通过这一功能，用户能够有效优化流量的使用，避免出现不必要的数据浪费。

2. 设置自定义规则

FAQ

VMess 使用 WebSocket 是否有安全优势？
WebSocket 可伪装成普通 HTTP/HTTPS 流量，配合 TLS 加密，可隐藏真实通信特征，提升抗封锁能力。结合策略组使用，可保证访问速度和网络稳定性，同时增加节点安全性。
如何使用 Sing-box 配合 Clash Rules 实现精细分流？
将 Clash 的 rules 段提取出来转换为 Sing-box 的 routing 格式，或直接引用 rule-providers 源。Sing-box 的路由引擎支持更复杂的条件匹配，可结合 GeoIP 与 keyword 实现精细分流策略。
Sing-box 如何实现精确端口分流？
Sing-box 支持按端口匹配流量，将不同类型应用流量分配到指定节点。结合策略组，可提高访问速度和网络稳定性，实现精准分流和优化用户体验。
Mellow TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响指定应用。TUN 更适合全局加速，如游戏、视频和浏览器流量，提高网络稳定性和覆盖范围。
Mihomo 是什么？
Mihomo 是 Clash Premium 的开源替代项目，兼容原版 Clash 配置文件并扩展了部分新功能。它性能优异、内存占用低，适用于希望自定义构建代理环境的高级用户。
PassWall2 如何结合节点测速优化策略组？
PassWall2 会定期测速节点延迟和丢包率，并根据策略组自动选择最优节点。通过测速和自动切换，可保证网页、视频和游戏流量走低延迟节点，提升整体网络体验。
PassWall2 的“节点切换策略”如何配置？
在 PassWall2 中可设置多节点策略，如“按延迟排序”、“自动选择最快节点”等。程序会定期测速，根据实时数据自动切换线路，保证连接的稳定性和速度。
Vmess 的安全加密方式推荐使用哪种？
目前推荐使用 auto 或 none（结合 TLS 加密）以提升性能。旧版 AEAD 加密已被废弃，使用 TLS/XTLS 层提供的加密即可确保安全。过度加密反而增加 CPU 占用与延迟。
WinXray 的测速功能如何使用？
WinXray 内置节点测速工具，可以通过延迟测试（ping）或带宽测试判断节点质量。用户只需右键节点选择“测速”即可，测试结果将显示在列表中，帮助快速挑选最佳线路。
VMess 使用 AEAD 加密对安全性有何帮助？
AEAD 加密可防止数据篡改、重放攻击和流量特征泄露。高封锁环境下使用，有助于保持节点长期可用，保证数据安全和连接稳定性，同时降低被检测的风险。
