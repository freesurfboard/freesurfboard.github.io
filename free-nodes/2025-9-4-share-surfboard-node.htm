示例规则实现：

4.2 带宽管控黑科技

通过TC命令实现QoS：
bash tc qdisc add dev eth0 root handle 1: htb default 20 tc class add dev eth0 parent 1: classid 1:1 htb rate 100mbit tc class add dev eth0 parent 1:1 classid 1:10 htb rate 30mbit ceil 50mbit tc filter add dev eth0 protocol ip parent 1:0 u32 match ip dst 192.168.1.100 flowid 1:10 这将为IP为192.168.1.100的设备限制最大50Mbps带宽。
